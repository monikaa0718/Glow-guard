# 🌸 Glow Guard

An AI-powered skincare ingredient analyzer that helps users make informed skincare decisions.

## 🚀 Live Demo

🔗 https://glow-guard-topaz.vercel.app/

## 📖 About

Glow Guard enables users to analyze cosmetic ingredients based on their skin type. By uploading a product label or entering ingredients manually, users receive personalized safety scores, ingredient insights, and skincare recommendations.

## ✨ Features

* 📸 Upload cosmetic product labels
* ✍️ Enter ingredients manually
* 👩 Select skin type (Normal, Oily, Dry, Combination, Sensitive)
* 🛡️ AI-powered ingredient analysis
* 📊 Personalized safety scores
* 🔍 Detailed ingredient insights
* 💡 Skincare recommendations
* 📱 Responsive user interface

## 🛠️ Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Supabase
* Vercel

## 🎯 Use Case

Choosing skincare products can be confusing due to complex ingredient lists. Glow Guard simplifies this process by analyzing ingredients and providing personalized insights tailored to different skin types.

## 📂 Installation

```bash
git clone https://github.com/monikaa0718/Glow-guard.git
cd Glow-guard
npm install
npm run dev
```
